# Next.js Application

Complete Next.js application with CI/CD pipeline.

## Features

- ✅ Next.js 14 with App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ API Routes
- ✅ Jest Testing
- ✅ PM2 Process Management
- ✅ GitHub Actions CI/CD
- ✅ Vercel Deployment

## Development
```bash
npm install
npm run dev
```

## Production
```bash
npm run build
npm start
```

## Testing
```bash
npm test
```

## Deployment

Automatically deploys to Vercel on push to main branch.
